# 03-模块划分

## 核心模块

### 1. 任务调度模块 (task/)

负责XXL-Job任务定义和执行

**主要类:**
- `CommunityJob` - 社区相关任务
- `FirebaseTask` - Firebase相关任务
- `KlaviyoTask` - Klaviyo同步任务

### 2. 服务层 (service/)

业务逻辑实现

**主要服务:**
- `FoodScaleReportService` - 食物秤报表
- `DeviceReportService` - 设备报表
- `CommunityService` - 社区服务
- `DingTalkService` - 钉钉通知

### 3. 数据访问层 (mapper/)

MyBatis数据访问

**映射器:**
- `*Mapper.java` - 接口定义
- `mapper/*.xml` - SQL映射

### 4. 领域模型 (domain/)

数据模型定义

**类型:**
- `vo/` - 值对象
- `dto/` - 数据传输对象
- `enums/` - 枚举定义

### 5. 集成模块

| 模块 | 路径 | 功能 |
|------|------|------|
| Zendesk | `zendesk/` | 工单系统集成 |
| Klaviyo | `kalaviyo/` | 营销自动化集成 |
| Robot | `robot/` | 客服机器人 |

## 模块依赖关系

```
task/ → service/ → mapper/
          ↓
       domain/
          ↓
      config/
```

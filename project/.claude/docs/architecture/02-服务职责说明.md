# 02-服务职责说明

## 核心职责

作为分布式任务调度执行器，负责执行各类后台定时任务

## 主要任务类型

### 1. 数据报表任务

| 任务 | 说明 |
|------|------|
| FoodScaleReportTask | 食物秤数据日报/月报 |
| DeviceReportTask | 设备报表 |
| CommunityReportTask | 社区数据统计 |

### 2. 数据同步任务

| 任务 | 说明 |
|------|------|
| UserSyncTask | 用户数据同步 |
| KlaviyoSyncTask | Klaviyo事件同步 |
| ElasticsearchSyncTask | ES数据同步 |

### 3. 第三方集成任务

| 任务 | 说明 |
|------|------|
| FirebaseTask | Firebase推送/分析 |
| DingTalkTask | 钉钉通知 |
| ZendeskTask | Zendesk工单同步 |

### 4. 数据处理任务

| 任务 | 说明 |
|------|------|
| VideoConvertTask | 视频转码 |
| EmailValidateTask | 邮件验证 |
| BadgeTask | 勋章数据处理 |

## 数据源切换

使用 `@DS("datasourceName")` 注解切换数据源:

```java
@DS("sys")      // 主数据库
@DS("click")    // ClickHouse
@DS("fit")      // fit数据库
@DS("food-scale") // 食物秤数据库
```

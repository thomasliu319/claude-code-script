# 04-调用关系

## 整体调用链

```
XXL-Job Admin
    │
    ▼
XXL-Job Executor (schedule-tasks:9997)
    │
    ├──▶ Task类 (@Component + @XxlJob)
    │         │
    │         ▼
    │     Service层
    │         │
    │         ▼
    │     Mapper层 (数据访问)
    │         │
    │         ▼
    │     多数据源 (@DS注解)
    │
    ├──▶ 第三方服务
    │    ├── Firebase
    │    ├── Klaviyo
    │    ├── Zendesk
    │    ├── DingTalk
    │    └── AWS S3
    │
    └──▶ 消息通知
         ├── 钉钉机器人
         └── 邮件/推送
```

## 数据流

```
1. 查询数据
   Query → Mapper → @DS → Database

2. 处理数据
   Process → Service Business Logic

3. 生成报表
   Generate → EasyExcel → Local File

4. 上传存储
   Upload → AWS S3 → Get URL

5. 发送通知
   Notify → DingTalk Webhook → User
```

## 关键接口

| 接口 | 位置 | 说明 |
|------|------|------|
| `XxlJobExecutor` | xxL-Job框架 | 任务执行入口 |
| `XxlJobHelper` | XXL-Job框架 | 任务辅助工具 |
| `AwsS3Util` | utils | S3文件上传 |
| `DingTalkUtil` | utils | 钉钉通知 |

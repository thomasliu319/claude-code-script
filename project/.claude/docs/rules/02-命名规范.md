# 02-命名规范

## Java命名规范

### 类命名

| 类型 | 规范 | 示例 |
|------|------|------|
| Controller | `*Controller` | `UserController` |
| Service接口 | `*Service` | `UserService` |
| Service实现 | `*ServiceImpl` | `UserServiceImpl` |
| Task类 | `*Task` | `FoodScaleReportTask` |
| Job类 | `*Job` | `CommunityJob` |
| Mapper接口 | `*Mapper` | `UserMapper` |
| VO | `*VO` | `UserVO` |
| DTO | `*DTO` | `UserDTO` |
| 枚举 | `*Enums` | `ZendeskTicketStatusEnums` |
| 常量类 | `*Constant` | `ZendeskUrlConstant` |
| 格式化器 | `*Formatter` | `ModelFormatter` |

### 方法命名

| 用途 | 规范 | 示例 |
|------|------|------|
| 查询 | `get*` / `query*` | `getUserById`, `queryUsers` |
| 新增 | `insert*` / `add*` | `insertUser`, `addOrder` |
| 更新 | `update*` | `updateUser` |
| 删除 | `delete*` | `deleteUser` |
| 定时任务方法 | `execute*` | `executeDailyReport` |

## 包命名

- **配置类**: `com.renpho.*.config`
- **领域对象**: `com.renpho.*.domain`
- **服务实现**: `com.renpho.*.service.impl`
- **任务类**: `com.renpho.*.task`
- **工具类**: `com.renpho.*.utils`

## 数据库命名规范

### 表命名

- 系统表: `sys_*`
- 业务表: 对应业务模块

### 字段命名

- 使用下划线分隔: `user_id`, `created_at`
- 布尔类型: `is_*`, `has_*`

## XXL-Job任务命名

```
{beanName}.{methodName}
例如: foodScaleReportTask.executeDailyReport
```

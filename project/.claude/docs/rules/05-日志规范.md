# 05-日志规范

## 日志记录原则

### 必须记录日志的场景

1. **任务开始/结束**
   ```
   logger.info("=== {} 任务开始 ===", taskName);
   logger.info("=== {} 任务结束, 处理 {} 条数据 ===", taskName, count);
   ```

2. **异常错误**
   ```java
   logger.error("执行失败: {}", errorMsg, e);
   ```

3. **关键业务节点**
   - 数据查询完成
   - 文件上传/下载
   - 第三方调用

### 日志格式规范

```
[时间] [级别] [类名] - 消息
例如: 2024-01-01 10:00:00 INFO FoodScaleReportTask - 任务开始
```

## 日志变量

| 变量 | 说明 |
|------|------|
| `{}` | 占位符，自动调用toString() |
| `{}` | 多参数按顺序填充 |

## 敏感信息处理

- **禁止**记录: 密码、Token、信用卡号
- **脱敏处理**: 手机号、邮箱等个人隐私

## XXL-Job日志

通过 `XxlJobHelper.log()` 输出到Job Admin控制台

# 04-异常处理规范

## 异常分类

### 系统异常

- 数据库连接失败
- Redis连接异常
- XXL-Job执行异常

### 业务异常

- 数据不存在
- 业务校验失败
- 第三方服务调用失败

## 异常处理方式

### 1. 捕获异常并记录日志

```java
try {
    // 业务逻辑
} catch (Exception e) {
    logger.error("执行失败: {}", e.getMessage(), e);
    XxlJobHelper.handleFail("任务执行失败: " + e.getMessage());
}
```

### 2. XXL-Job任务异常处理

使用 `XxlJobHelper.handleFail()` 标记任务失败

### 3. 返回值处理

- 查询为空: 返回空列表 `Collections.emptyList()`
- 操作失败: 返回 `Boolean.FALSE`

## 日志级别使用

| 级别 | 使用场景 |
|------|----------|
| ERROR | 系统错误、业务异常 |
| WARN | 潜在问题 |
| INFO | 关键流程节点 |
| DEBUG | 调试信息 |

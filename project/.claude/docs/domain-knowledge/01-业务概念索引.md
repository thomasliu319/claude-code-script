# 01-业务概念索引

## 业务领域

### 1. 设备相关 (Device)

| 概念 | 枚举 | 说明 |
|------|------|------|
| 设备类型 | `DeviceTypeEnum` | 体重秤、食物秤、血压计、血氧仪等 |
| 设备关联 | `DeviceTypeRelationEnum` | 设备关联关系 |
| 设备报表类型 | `DeviceReportType` | 日报、月报、新增统计 |

**设备类型 (DeviceTypeEnum):**
```
SCALE - 体重秤
FOOD_SCALE - 食物秤
SPHYGMOMANOMETER - 血压计
BLOOD_OXYGEN - 血氧仪
TREADMILL - 跑步机
RING - 智能戒指
```

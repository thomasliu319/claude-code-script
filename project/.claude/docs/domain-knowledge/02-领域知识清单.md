## 第三方集成

### 1. Firebase
- **用途**: 推送通知 + 分析数据
- **SDK**: Firebase Admin SDK

### 2. Klaviyo
- **用途**: 营销自动化事件同步
- **事件类型**: 用户测量、设备使用、社区活动等

### 3. Zendesk
- **用途**: 工单系统集成
- **功能**: 工单创建、状态同步、自定义字段

### 4. DingTalk
- **用途**: 内部通知
- **方式**: 机器人Webhook

### 5. AWS S3
- **用途**: 文件存储（报表、日志等）
- **工具类**: `AwsS3Util`

## 报表生成流程

```
1. 数据查询 → Mapper → 多数据源
2. 数据处理 → Service 业务逻辑
3. Excel生成 → EasyExcel → 本地文件
4. 文件上传 → AWS S3 → 获取URL
5. 通知发送 → 钉钉机器人 → 通知相关人
```
